## Alien Invasion Game

Pre-requisites:
- python3

One Command to Run:
(just copy/paste all in a terminal)
```
git clone https://github.com/shubmehetre/alien_invasion_game.git
cd alien_invasion_game
python3 alien_invasion.py
```

How to play:
- Play: p
- Shoot: spacebar
- Move: arrow keys
- Quit: q
